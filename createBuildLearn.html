  • A: In Create - Build - Learn (CBL) we'll learning about Amazon Web Services (AWS) tools and resources to create projects using
    • Sumerian to learn about AR/VR
    • RoboMaker to learn about Robotics

  • Q: Who is hosting and teaching Create - Build - Learn?
  • A: Hocking College will be hosting the classes at their campus in Nelsonville Ohio. Tasha Penwell is teaching the classes on Mondays from 5:30p - 6:30p. Class attendance is optional and not required to earn digital badges
  • Q: When is Create - Build - Learn?
  • A: We will be hosting classes in person on Mondays 5:30pm - 6:30pm. The curriculum is 100% online and class attendance is optional.   • Sumerian (Feb 3, 10, 17, 24)
  • RoboMaker (Mar 2, 9, 16, 23, 30)
  • IoT (Apr 6, 13, 20, 27)
